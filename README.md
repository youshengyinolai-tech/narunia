# ナルニア国物語 期末試験対策ツール

『ライオンと魔女』（第1〜5章）の授業教材（対訳・語彙マーカー・下線部）を使った、自分専用の期末試験対策アプリです。単一のHTMLファイルとして動作し、ビルド不要でブラウザで開くだけで使えます。

## 使い方

`narnia_study_tool.html` をブラウザで開くだけです。

- **単語・熟語**:
  - カード: 語彙フラッシュカード（Leitner式間隔反復つき、タップで日本語訳を表示）
  - 一覧: 全語彙を検索できるリスト表示
  - マッチ: 英語と意味のタイルをペアで選んで消していくQuizletスタイルのゲーム
  - 「小テストの語のみ」フィルタで、授業の小テストに出た語だけに絞って学習可能
- **下線部訳**: 下線が引かれた語句の和訳練習
- **読解クイズ**:
  - 日本語訳を選ぶ: 4択の翻訳問題
  - 内容理解: 本文の出来事・事実を問う4択問題（「長女の名前は？」など）
- **文法問題**: 本文の実際の文をもとにした文法4択問題（文法ポイントは初期状態では隠れており、ヒントボタンで表示）
- **小テスト対策**: 授業の小テストで実際に出た穴埋め問題（約80問）をそのまま4択で復習
- **まとめ**: 章ごとのあらすじ・重要文法・重要語彙と、登場人物紹介（イラストつき、10人）
- **復習**: クイズ・内容理解・下線部訳・文法問題・小テスト対策で間違えた問題を自動収集して再出題

進捗はブラウザの`localStorage`に保存され、次回開いたときも引き継がれます。

## パイプライン（教材更新時の再生成）

`pipeline/` ディレクトリに、元のWord教材（docx）から `narnia_study_tool.html` を再生成するスクリプト一式が入っています。詳細は [`pipeline/README.md`](pipeline/README.md) を参照してください。
